# My Rust Tools Scoop Bucket

This is a custom [Scoop](https://scoop.sh/) bucket for my Rust tools.

## How to use

If you haven't already, [install Scoop](https://scoop.sh/#install).

Then, add this bucket:

```powershell
scoop bucket add <bucket-name> https://github.com/<user>/scoop-bucket
```

Now you can install the tools:

```powershell
scoop install my-rust-tool
```
